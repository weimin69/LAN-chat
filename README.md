# LAN Chat Room

一个基于 Python Socket 的局域网聊天室项目。

适合：

- Python 初学者
- 网络编程入门
- Socket 学习
- 多线程实践

---

## 项目介绍

这是一个运行在 **局域网（LAN）** 中的聊天室。

多个用户可以在同一个网络下互相发送消息。

采用：

> Client - Server（客户端 / 服务器）架构

所有客户端连接到一个服务器，由服务器负责转发消息。

项目目标：

- 学习 Socket 编程
- 理解 TCP 通信
- 学习多线程
- 理解客户端与服务器架构

## 快速开始

### 环境要求

- Python 3.6+
- 纯标准库，无需安装任何第三方依赖

### 1. 启动服务器

在作为服务器的机器上运行：

```bash
python server.py
```

启动后显示 `服务器启动`，等待客户端连接。

### 2. 启动客户端

在任意局域网内的机器上运行：

```bash
python client.py
```

输入服务器的 IP 地址后即可加入聊天室。

> 查看服务器 IP：在服务器机器上运行 `ipconfig`（Windows）或 `ifconfig`（Mac/Linux）。

### 3. 开始聊天

多个客户端连接后，任意客户端发送的消息会被广播给所有其他客户端。

## 工作原理

```
客户端A  ──msg──▶  服务器  ──转发msg──▶  客户端B
                    │
                    └──转发msg──▶  客户端C
```

- **服务器** (`server.py`)：绑定 `0.0.0.0:9999`，监听客户端连接。收到消息后广播给所有其他客户端。使用多线程为每个客户端独立收发。
- **客户端** (`client.py`)：连接到服务器，开启接收线程监听消息，主线程负责发送用户输入。
